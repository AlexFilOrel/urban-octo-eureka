## Списки

Чтобы добавить маркированные списки, необходимо пункты выделить звездочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты пронкмеровать. Например, вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
